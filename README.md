# DeckSide Releases

Public release artifacts (Windows installer + electron-updater feeds) for [DeckSide](https://deckside.app), the swim-meet day desktop app for EBSL coaches.

- **Latest release**: see [Releases](https://github.com/sportside-app/deckside-releases/releases)

This repo only hosts the compiled `.exe` installer plus electron-updater metadata files (`latest.yml` / `latest-beta.yml` / `latest-alpha.yml`). Auto-update points here so installed clients can fetch updates without any GitHub token.
